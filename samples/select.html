Single select with viewport-aware edge detection.

`bp-ui-select` is the shared select contract for future projects: stable HTML, bubbling events, fixed-position popovers, and edge-aware placement that keeps the option list visible near viewport boundaries.

Single-select only Above/below flip Horizontal clamping

Behavior split

The vanilla controller owns viewport-aware popover placement. The Alpine adapter reuses the same option state and selection rules when the wrapper wants to manage placement itself.

Vanilla snippet

Copy-ready markup for the stable select contract.

<div class="bp-ui-select" data-bp-ui-select data-placeholder="Choose a property type" data-name="propertyType">
  <div class="bp-ui-select__label">Property Type</div>
  <button class="bp-ui-select__trigger" type="button" aria-haspopup="listbox">
    <span class="bp-ui-select__trigger-value">Choose a property type</span>
    <span class="bp-ui-select__trigger-chevron" aria-hidden="true"></span>
  </button>
  <input class="bp-ui-select__input" type="hidden" />
  <div class="bp-ui-select__popover" role="listbox">
    <button class="bp-ui-select__option" type="button" data-value="villa">
      <span class="bp-ui-select__option-label">Villa</span>
      <span class="bp-ui-select__option-indicator" aria-hidden="true"></span>
    </button>
    <button class="bp-ui-select__option" type="button" data-value="cabin">
      <span class="bp-ui-select__option-label">Cabin</span>
      <span class="bp-ui-select__option-indicator" aria-hidden="true"></span>
    </button>
  </div>
</div>

Alpine snippet

Copy-ready adapter example with the same viewport-aware placement as the vanilla controller.

<div
  class="relative rounded-[1.75rem] border border-slate-200 bg-white p-5 shadow-sm"
  x-data="window.bpUISelect()"
  x-init="init()"
>
  <div class="bp-ui-select" data-value="garden">
    <div class="bp-ui-select__label">Alpine View</div>
    <button class="bp-ui-select__trigger" type="button" @click="toggle()">
      <span class="bp-ui-select__trigger-value">Garden</span>
      <span class="bp-ui-select__trigger-chevron" aria-hidden="true"></span>
    </button>
    <input class="bp-ui-select__input" type="hidden" value="garden" />
    <div class="bp-ui-select__popover" hidden>
      <button class="bp-ui-select__option" type="button" data-value="ocean" @click="select($event.currentTarget.dataset.value, 'option'); closePopover()">
        <span class="bp-ui-select__option-label">Ocean</span>
        <span class="bp-ui-select__option-indicator" aria-hidden="true"></span>
      </button>
      <button class="bp-ui-select__option is-selected" type="button" data-value="garden" @click="select($event.currentTarget.dataset.value, 'option'); closePopover()">
        <span class="bp-ui-select__option-label">Garden</span>
        <span class="bp-ui-select__option-indicator" aria-hidden="true"></span>
      </button>
      <button class="bp-ui-select__option" type="button" data-value="city" @click="select($event.currentTarget.dataset.value, 'option'); closePopover()">
        <span class="bp-ui-select__option-label">City</span>
        <span class="bp-ui-select__option-indicator" aria-hidden="true"></span>
      </button>
    </div>
  </div>
</div>
